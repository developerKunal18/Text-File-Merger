# Text File Merger

A Python script that merges two text files into one output file.

## Description
This project reads content from two text files and combines them into a new file.

## Features
- Reads multiple files
- Writes merged output
- Simple file handling logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Place text files in the project folder
3. Run:

   python file_merger.py

4. Enter file names when prompted

## Example

Input:
file1.txt
file2.txt
output.txt

Output:
Files merged successfully.
